# AI Personal Assistant

## Project Overview

AI Personal Assistant is a full-stack Generative AI project built using Flask and Django. It helps users perform productivity tasks such as chatting with an AI assistant, summarizing notes, creating reminders, and generating productivity plans.

## Features

- Conversational AI Chat Interface
- AI Reminder Assistant
- AI Note Summarizer
- AI Productivity Planner
- Personalized AI Responses
- Flask API Integration
- Django Backend Integration
- SQLite Database

## Tech Stack
